# GitHub Pages Checklist

## Før du starter
- [ ] Git er installeret på din computer
- [ ] Du har en GitHub konto
- [ ] Du har oprettet en ny repository på GitHub

## I din projektmappe
- [x] `index.html` er i roden af mappen
- [x] `.gitignore` fil er oprettet
- [ ] Git repository er initialiseret (`git init`)
- [ ] Alle filer er tilføjet (`git add .`)
- [ ] Første commit er lavet (`git commit -m "..."`)
- [ ] Remote repository er tilføjet (`git remote add origin ...`)
- [ ] Koden er pushet til GitHub (`git push -u origin main`)

## På GitHub
- [ ] Repository er sat til "Public"
- [ ] GitHub Pages er aktiveret under Settings → Pages
- [ ] Source er sat til `main` branch og `/ (root)` folder
- [ ] Du har ventet 2-10 minutter efter første push

## Test
- [ ] Besøg `https://DIT-BRUGERNAVN.github.io/DIT-REPOSITORY-NAVN/`
- [ ] Siden loader korrekt
- [ ] Billeder vises korrekt
- [ ] Links virker

## Almindelige fejl
❌ Repository er private (skal være public for gratis GitHub Pages)
❌ Forkert branch navn (skal være `main` eller `master`)
❌ `index.html` er i en undermappe i stedet for roden
❌ Filer er ikke pushet til GitHub
❌ GitHub Pages er ikke aktiveret i Settings
